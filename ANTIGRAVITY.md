# AntiGravity 專案協作設定

這個檔案記錄了 AI 助手（Antigravity）在此專案中的協作規則、專案設定與目前的開發狀態。

## 專案基本資訊

- **專案名稱**：ExtremeSignalAppCS (Trading-strategy-CS-2)
- **專案類型**：C# WPF 應用程式 (極值觀測與交易策略程式)
- **主要串接 API**：元大交易 API (Yuanta API)
- **Git 管理**：已建立本地 Git 倉庫，目前僅需本地 Git 管理（無 GitHub 遠端 CLI 同步需求）
- **部署需求**：無部署需求（為本地運行的交易程式/GUI）

## 開發與協作原則

1. **輸出語言**：所有輸出（包含回覆、思考、任務清單、分析、建議、程式碼註解）均使用**繁體中文**，嚴禁使用簡體或其他語言。
2. **謹慎操作**：在修改現有程式碼前，應先分析邏輯，避免破壞現有的元大 API 串接與交易引擎核心。
3. **還原指令規則**：
   - 當使用者說「切換還原至第一版」時，AI 應協助執行以下 Git 指令將工作目錄強制還原到標籤 `第一版`：
     ```bash
     git reset --hard 第一版
     ```
   - **注意**：在執行此指令前，必須先提醒使用者這會捨棄所有目前未提交的修改，並確認其是否要繼續。
