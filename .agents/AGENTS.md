# 專案特定規則 (Workspace Rules)

- 當使用者說「切換還原至第一版」時，請協助執行以下 Git 指令將工作目錄強制還原到標籤 `第一版`：
  ```bash
  git reset --hard 第一版
  ```
  **注意**：在執行此指令前，必須先提醒使用者這會捨棄所有目前未提交的修改，並確認其是否要繼續。

- 當使用者說「切換還原至第二版」時，請協助執行以下 Git 指令將工作目錄強制還原到標籤 `第二版`：
  ```bash
  git reset --hard 第二版
  ```
  **注意**：在執行此指令前，必須先提醒使用者這會捨棄所有目前未提交的修改，並確認其是否要繼續。
